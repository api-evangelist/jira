# Jira

> Atlassian Jira project management and issue-tracking platform. This index links the Jira API surface
> — Cloud Platform REST (v3/v2), Jira Software (Agile), Jira Service Management, JSM Operations, Jira
> Align, and Customer Service Management — plus the machine-readable artifacts API Evangelist has
> harvested and derived for it. Generated from the repository catalog; the docs host does not publish
> an llms.txt.

## APIs
- [Jira Cloud Platform REST API](https://developer.atlassian.com/cloud/jira/platform/): issues, projects, JQL search, transitions, comments (base https://your-domain.atlassian.net/rest/api/3)
- [Jira Cloud Platform REST API v2](https://developer.atlassian.com/cloud/jira/platform/rest/v2/intro/): v2 without Atlassian Document Format
- [Jira Software Cloud REST API](https://developer.atlassian.com/cloud/jira/software/): boards, sprints, backlogs (base /rest/agile/1.0)
- [Jira Service Management REST API](https://developer.atlassian.com/cloud/jira/service-desk/): queues, customers, requests, SLAs
- [Jira Service Management Operations REST API](https://developer.atlassian.com/cloud/jira/service-desk-ops/introduction/introduction/): schedules, on-call, alerts, incidents
- [Jira Align REST API](https://help.jiraalign.com/hc/en-us/sections/360008049974-API-2-0): portfolios, epics, program management
- [Customer Service Management REST API](https://developer.atlassian.com/cloud/customer-service-management/): customers, organizations, entitlements
