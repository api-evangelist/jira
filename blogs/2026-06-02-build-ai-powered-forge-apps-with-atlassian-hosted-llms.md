Atlassian announced that the Forge LLMs API is now in Preview release, enabling developers to integrate Claude models directly into Forge apps without external data transfer, keeping all data on Atlassian's infrastructure. The service allows developers to build AI-powered features like summarization and intelligent suggestions while maintaining enterprise security standards, with pricing based on a token-to-credit system. Three Claude model tiers are available to balance cost and capability for different use cases.
